# AGENTS.md

## 基本方針

- 日本語で回答する。
- 記事を執筆する際は情報の信ぴょう性を重視し、ソースがあるものを使用する。

## 記事作成ルール

- ホテルを数えるときは「〇軒」ではなく「〇つ」を使う。「軒」も誤りではないが、文体が固くなりやすいため避ける。
- 正しい日本語を使う。主語・述語・目的語の対応が曖昧な文や、何がどうなるのか分からない表現を避ける。
- 「ずれやすい」「合わない」「失敗しやすい」などの表現を使う場合は、何がずれるのか、何と合わないのか、どう失敗しやすいのかを具体的に書く。
- 例: 「宮古島のホテルは、海の近さだけで選ぶとずれやすいです。」のような文は避ける。「宮古島のホテルは、海の近さだけで選ぶと、夕食や観光の移動が想定より不便になることがあります。」のように、ずれる内容を明確にする。
